一、KiCad入门使用

发表于2021-01-24|更新于2021-01-25
|阅读量:

我的第一个KiCAD项目，一个USB-TTL模块，芯片CH330N。由于是第一次用这个工具软件，内容比较细，多图预警:satisfied:

软件安装

官网下载安装。

创建工程

文件 -> 新建 -> 工程

绘制原理图

双击原理图文件，弹出绘制原理图工作窗口

配置一下信息

按快捷键A打开元件库，第一次使用时需要加载元件库，添加需要元器件

搜索框输入：

C，添加电容

R，添加电阻

LED，添加LED灯

CH330N，添加CH330芯片

USB_A，添加USB接口

Conn_01x05，添加排针

也可以选择“在符号库浏览器中选择”

按ESC键取消当前鼠标状态，将鼠标指针放到元件上按快捷键M移动位置，快捷键R翻转元件。

快捷键P增加电源

快捷键W布线，双击鼠标左键闭合布线。快捷键L怎讲网络标号。

给符号添加封装。

通过快捷键V或者双击元件文本修改。

两个电容修改为104，两个电阻修改为4.7K

最后给原理图做批准。

执行电器规则检查。

保存 -> 工具 -> 从原理图更新PCB

或者按F8。

绘制PCB

双击PCB文件。

右键 -> 网格 -> 1.0mm

在100，100处设置网格原点。

选择涂层。画边框

绘制25*16切割层，以及修改网格为0.1mm，画出缩小0.3毫米的电路编辑层。

放置元件，微调可通过M + 方向键,也可通过快捷键E修改坐标信息。

在原理图快捷键X``Y翻转引脚。

设置规则

布线，放置差分线(0.28mm)，电源线(1mm),其他(0.5mm)

快捷键D拉动布线，快捷键L锁定布线。V切换布线层和放置过孔。

覆铜

选择添加填充区域

第二面覆铜可以

右键 -> 覆铜 -> 重复填充，然后按B。

Alt+3看效果图
