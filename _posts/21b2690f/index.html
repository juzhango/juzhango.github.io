useradd

  • u: 新账户的用户 ID
  • c: 新账户的 GECOS 字段
  • s: 新账户的登录 shell

usermode

参考 id 为 1000 的用户，将新建的用户追加到与 1000 用户相同的组。

  • a: 将用户追加至上边 -G 中提到的附加组中，并不从其它组中删除此用户
  • G: 新的附加组列表 GROUPS
